populateFromJSON

any static populateFromJSON(any obj, string json)
Populate an object from a JSON object.

Populates the object 'obj' from the JSON properties recursively. If obj is an event, sets the fields from the JSON object.

Throws if the json contains a property that is not a field in an event of obj.
Parameters:
obj - The reference object to fill in properties of. Should be a sequence, dictionary or event.
json - the JSON object.
Returns:
the object (typically, the same as obj that was passed in)
