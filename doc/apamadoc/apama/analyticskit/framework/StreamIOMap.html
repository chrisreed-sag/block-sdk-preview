apama.analyticskit.framework
Event StreamIOMap

Map from StreamIO to a value.

StreamIO is like a dictionary with a StreamIO key. The partition field value is treated specially. In some methods (hasKey, getAll) a wildcard in either the key provided or an entry in the map will be treated as a match. add, remove methods require an exact match.
