apama.analyticskit.framework
Event ScenarioServiceUtil

Scenario service controller object.

This acts as a factory for ScenarioService requests and handles Edit and Delete requests as well. The monitor should call init, passing in action pointers for onSpawn and onEdit, and new instances of the monitor will be created as controlled by ScenarioService. This object keeps track of the scenario parameters, which can be obtained with the get methods.
