apama.analyticskit.framework.internal.parser
Event Token

Represents a lexical token.

A lexical token, which is one of an entire string literal (not used), identifier (e.g. 'input1'), number (possibly excluding leading '-'), operator, or a special type denoting end of 'file' (end of text). Whitespace is excluded by the parser.
