apama.analyticskit
Event Promise

An asynchronous promise result.

A promise represents something that may be fulfilled later. Promises can be chained, to make linear-style async chains of calls. The 'then' methods take an action which returns an any - if that any is a Promise, then it is waited on until any 'then' actions are called.

Promises are created with the static 'create' action, which takes an 'fn' action to call. This function is passed the fulfill and reject actions. (this pattern ensures that exceptions thrown by the 'fn' action are caught and reject the promise).
