Event L10N

API for localization of error message.

Blocks should throw getLocalizedException("msgId", [params]) (or the _basic equivalent), where params can be result of getLocalized(..) (or _basic variant). (the use of 'throw' in the block informs the compiler that that basic block is terminated).
